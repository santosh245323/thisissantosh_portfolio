OJT: Basic Computer Training

Introduction

This On-the-Job Training (OJT) is focused on foundational computer skills including hardware, software, and essential office tools. It is ideal for beginners aiming to start a career in IT support, administration, or general computer operations.

What You'll Learn

  • Computer hardware components and assembling
  • Installing and configuring Windows & Linux OS
  • Basic software installation and management
  • PC troubleshooting and maintenance
  • Networking basics (LAN, IP config, cabling)
  • Using MS Word, Excel, and PowerPoint efficiently
  • Email configuration and internet use
  • Printer/scanner installation and troubleshooting
  • Backup, formatting, and file management
  • Basic cybersecurity hygiene and updates

Final Output / Project

  • Complete a full PC setup from assembly to OS install
  • Simulate troubleshooting of hardware/software issue
  • Create and format documents, spreadsheets and presentations
  • Network two or more computers using LAN

Certification

Participants will receive a training certificate after successful completion of the OJT program, issued by the training institution. This adds value to your resume for entry-level IT jobs in both government and private sectors.
